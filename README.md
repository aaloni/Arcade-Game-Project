# Classic Arcade Game

## Instructions
- To run the game:
1- You need a browser (Google Chrome Recommended).
2- Open the Index.html file in "frontend-nanodegree-arcade-game-master" folder.

++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

- To play the game:
1- Use arrows to avoid the moving bugs.
2- Cross to the sea to win.

Happy Gaming!
